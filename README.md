## [项目简介](https://blog.csdn.net/ZhangYu971014/article/details/79841555)
项目是围绕电网规约101规约（DL/T634.5101-2002）和104规约（DL/T634.5104-2009），项目基于Java语言。
可以完成规约的内容解析工作和组装工作。可用于实际场景的把发送报文的生成等工作。
